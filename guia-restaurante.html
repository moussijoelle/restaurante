Guía del ejercicio — Restaurante Mamamia

Objetivo: entender HTML + JS + lógica con if.

Colores (para entender rápido):
🟨 ID 🟦 Buscar 🟩 Leer 🟥 Escribir
Ejemplo: buscar el botón por su id, leer lo que eligió la persona y escribir el mensaje final.

1) ¿En qué consiste el ejercicio?

Tienes 3 platos: carne, pasta (con gluten) y risotto (con lactosa).

La persona elige un plato (en un <select>) y marca sus preferencias/intolerancias (checkbox): vegano, intolerante a la lactosa, intolerante al gluten.

Cuando pulsa “validar el pedido”, el programa decide:

  • Puede comerlo → Perfecto, pedido correcto
  • No puede → Usted no puede comer este plato.

2) Tu HTML actual (copiado) y qué significa

Esto es un resumen de las partes importantes de tu index.html. 

<select id="operador">
  <option value="carne">carne</option>
  <option value="pasta">pasta (con gluten)</option>
  <option value="risotto">risotto (con lactosa)</option>
</select>

<input type="checkbox" id="vegano">
<input type="checkbox" id="no-lactosa">
<input type="checkbox" id="celiaco">

<button type="button" id="validar">validar el pedido</button>

<p id="mensaje"></p>

<script src="app.js"></script>

Explicación línea por línea (HTML)

  1. 🟨 ID id="operador": “nombre” del select. 🟦 Buscar JS lo encuentra con getElementById("operador").
  2. 🟩 Leer value="carne", value="pasta", value="risotto": es lo que JS compara en las reglas.
  3. 🟨 ID Cada checkbox tiene su id para leer .checked: true (marcado) o false (no marcado). 🟩 Leer
  4. 🟨 ID id="validar": JS escucha el click de este botón. 🟦 Buscar
  5. 🟥 Escribir id="mensaje": aquí JS escribe el resultado final.
  6. <script src="app.js">: carga el archivo JS.

3) Tu JavaScript actual (copiado) y qué significa

const boton = document.getElementById("validar");
const select = document.getElementById("operador");
const mensaje = document.getElementById("mensaje");
const plato = document.getElementById("operador").value;

const vegano = document.getElementById("vegano").checked;
const lactosa = document.getElementById("no-lactosa").checked;
const gluten = document.getElementById("celiaco").checked;

console.log(plato, vegano, lactosa, gluten);

boton.addEventListener("click", () => {
  const plato = select.value;

  const vegano = document.getElementById("vegano").checked;
  const lactosa = document.getElementById("no-lactosa").checked;
  const gluten = document.getElementById("celiaco").checked;

  let noPuede = false;

  if (plato === "carne" && vegano) noPuede = true;
  if (plato === "pasta" && gluten) noPuede = true;
  if (plato === "risotto" && lactosa) noPuede = true;

  if (noPuede) {
    mensaje.textContent = "Usted no puede comer este plato.";
  } else {
    mensaje.textContent = "Perfecto, pedido correcto";
  }
});

Explicación línea por línea (JS)

  1. 🟦 Buscar getElementById(...): busca un elemento por su 🟨 id.
  2. 🟩 Leer select.value: lee el plato elegido (carne, pasta, risotto).
  3. 🟩 Leer .checked: dice si una casilla está marcada (true) o no (false).
  4. 🟦 Buscar addEventListener("click", ...): ejecuta el código cuando se pulsa el botón.
  5. noPuede empieza en false. Si alguna regla se cumple, pasa a true.
  6. Reglas:
    • carne + vegano → no puede
    • pasta + gluten → no puede
    • risotto + lactosa → no puede
  7. 🟥 Escribir Al final: si noPuede es true → escribe “no puede”; si es false → escribe “Perfecto...”.
